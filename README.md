# Tutorial 7 - Hamiltonian Monte Carlo (HMC)

This project contains the Python code and LaTeX report for SOFE 4820U Modelling and Simulation – Tutorial 7 on Hamiltonian Monte Carlo.

## Files Provided

- `tutorial7_hmc.py`: A standalone Python script that implements the 1D HMC sampler, computes acceptance rates, and exports the sampler states to `.npy` arrays inside `data/`.
- `generate_figures.py`: A script that consumes the simulated `.npy` states to generate normalized sample histograms and the 2D Gaussian conceptual visual plot.
- `tutorial7_report.tex`: The LaTeX source code for the report, containing answers to all conceptual and experimental questions.
- `figures/`: The directory containing necessary figure outputs generated by `generate_figures.py`.
- `data/`: The directory containing raw saved data generated by `tutorial7_hmc.py`.
- `README.md`: This file.

## Requirements

Ensure the following packages are installed to run the Python scripts:
- `numpy`
- `matplotlib`
- `scipy`

You can install them via pip if necessary:
```bash
pip install numpy matplotlib scipy
```

## Running the Python Scripts

To separate the heavy simulation computation from visual generation, the project is run in two stages.

First, run the simulation algorithm:
```bash
python tutorial7_hmc.py
```
This script computes the HMC steps and writes the sample arrays (`samples_025.npy` and `samples_08.npy`) into the `data/` directory.

Second, run the figure generation tool:
```bash
python generate_figures.py
```
This parses the data and will safely generate all the required PNG figures inside the `figures/` directory.

## Compiling the LaTeX Report

To compile the `tutorial7_report.tex` into a PDF format, use your LaTeX compiler of choice. If you are using `pdflatex` from the terminal, ensure that the `figures/` directory is populated first by running the Python scripts, and then execute:

```bash
pdflatex tutorial7_report.tex
```

You may need to run this command a second time to properly link any embedded references.
